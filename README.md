Build Requirements
------------------

- OpenDiamond
- glib2
- libarchive


Building
--------

1. `make`.  To override MATLAB path, pass `MATLAB_ROOT_DIR`, `MATLAB_LIBDIR`,
and/or `MATLAB_EXE_PATH`.

2. `make install`.  To install into different directories, pass `BINDIR`
and/or `FILTER_DIR`.


Running
-------

MATLAB must be installed on the Diamond servers.
